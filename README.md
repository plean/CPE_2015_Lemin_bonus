# CPE_2015_Lemin_bonus

Bonus part of Lemin (labyrinth resolution algorithm) wich display the solving process.
This bonus was made using [Liblapin](https://github.com/Damdoshi/LibLapin).

### Usage

`lemin output | ./bonus`
